## Technology Stack

This website is built with:
- **Frontend:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Deployment:** Docker + Nginx
- **CI/CD:** GitHub Actions
- **Infrastructure:** VPS with automated deployments
